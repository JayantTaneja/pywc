# pywc
A python implementation of unix wc tool


### Installation

```bash
git clone https://github.com/JayantTaneja/pywc.git
cd pywc
pip3 install -e .
```

---

### Usage

Pretty much the same as unix wc tool ``` pywc <filename> [flags]``` . Following are the supported flags :

```bash

-c  : Byte Count
-w  : Word Count
-l  : Newline Count

```
